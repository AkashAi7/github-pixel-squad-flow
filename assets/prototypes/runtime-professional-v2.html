UI control changes

  • Remove the large top-level “Talk to <agent>” button from the hero.
  • Keep one subtle “Ping lane” action in the inspector for the selected agent.
  • On agent cards, make the action secondary-sized and visible on hover or selection.
  • Prefer “Ping lane” or “Continue lane” over a loud global talk CTA.

Design Direction

Industrial mission control

Professional, not playful-basic

Dark glass panels, amber/cyan signal colors, dense but readable metrics, and restrained motion. The UI should feel like an operations console, not a generic SaaS dashboard.
